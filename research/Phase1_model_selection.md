# Phase 1: Model Selection Result

Seven models were evaluated for phishing detection.

The Hybrid Linear SVM achieved the best overall performance:

- Accuracy: 99.40%
- Precision: 99.15%
- Recall: 99.78%
- F1 Score: 99.47%

Therefore, Hybrid Linear SVM is selected as the proposed model for PhishScope.

## Research Finding

Rule-only models showed weaker performance, while text-based and hybrid models performed significantly better.

This supports the research direction that phishing detection benefits from combining semantic text classification with explainable phishing indicators.
